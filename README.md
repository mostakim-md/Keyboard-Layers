# Ergonomic Keyboard Mods
Modern keyboards are still using the century old type writer layout. Thus a growing number of people are suffering from wrist pain. The best solution for this problem is split keyboard and the second best solution is customizing your pre-existing keyboard in a more ergonomoic way through software. 
This repo contains code for ahk, kanata code for ergonomic keyboard mods. Though ahk is more popular, I would recommend using Kanata as it is easier to customize.
